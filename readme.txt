Add the new comment subscription option to the blog comment form.

== Description ==

Comment Notifier lets user to subscribe the comment thread of each single post of your blog. It improves traffic and loyalty.

Notification emails are fully customizable (both plain text or html) with a number of "tags" to insert author name, post link and title, receiver name and so on.

The "subscribe me" checkbox in the comment form area can be added automatically or manually.

Read more on [Comment Notifier](http://www.satollo.net/plugins/comment-notifier) offical page.

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
